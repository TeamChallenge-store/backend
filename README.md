# Installation
 
✔️ Install dependencies:     
```bash
$ git clone https://github.com/0leksandrr/team_challenge.git 
```
✔️ Install dependencies:
```bash 
$ pip install -r requirements.txt
```
✔️ Apply migrations: 
```bash
$ python manage.py migrate
```
✔️ Run the server
```bash
$ python manage.py runserver
```
